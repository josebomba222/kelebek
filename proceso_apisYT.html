PROCESO CREACION API KEYS PERSONALES


Por el usuario del foro de Kodi jmh2002 (Traducción by Anabel)

* Paso 1:
- Entra aquí - https://console.cloud.google.com/
- Selecciona tus preferencias de Email (Sip o Nop, lo mismo da que da lo mismo)
- Haz click en Yes (que estás de acuerdo con los Términos del Servicio (Por si estás muy aburrido y quisieras leerlos: https://console.cloud.google.com/terms)
- Haz click en Agree y Continue

Nota: Ignora cualquier oferta tipo "Sign up for a free trial..." y/o "Try Google Cloud Platform for free". No es necesario para nada

- Sigue estas opciones: Select Project > Create Project > Enter Project Name (Pon el nombre que te de la gana) > Create
- ESPERA hasta que la siguiente página se cargue con el Proyecto que acabas de crear.







* Paso 2:

- Ahora entra aquí- https://console.developers.google.com/apis/library
- Y sigue estos pasos: YouTube APIs > Select YouTube Data API > Enable





* Paso 3:

- Ahora entra aquí - https://console.cloud.google.com/apis/credentials
- Y escoge: Create Credentials > API Key
- Copiate la API Key, la necesitarás más tarde. Es privada, no se la des a nadie.
- Y ya puedes cerrar.





* Paso 4:

- Ahora, en la misma página escoge: Create Credentials > OAuth Client ID
- Y ahora esto: Select > Configure Consent Screen





* Paso 5:

- Mete un nombre (el que te dé la real gana) en "Product name shown to users"
- Click en Save and ESPERA que se cargue la siguiente página con el OAuth Client ID screen
- Selecciona: Other > Create
- Copiáte el Client ID y el Client Secret. También los necesitarás más tarde. Y no se los des a nadie.
- Clikea ahora en OK.





* Paso 6:

- Ahora nos vamos al addon Youtube > Ajustes > API > Habilitamos las API Keys Personales > Metemos las 3 Keys > Y OK
- Nos salimos y entramos de nuevo (con las normales 2 x web activation)

Una vez que nos identifiquemos con éxito en el addon YouTube, ya deberíamos estar usando las API Keys que has creado.
