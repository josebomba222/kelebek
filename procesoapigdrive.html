PROCESO CREACION API KEYS GDRIVE PERSONALES



Lo primero, tranquilidad, que puede parecer complicado pero son en realidad 6 pasitos de nada.
Además, que serán 6 pasitos para ti pero un gran paso para el addon Youtube. Así que venga, despacito y buena letra...



Recomendación de la traductora: Yo haría el proceso en un PC, para así poder usar el CTRL-C y CTRL-V a la hora de colocar los 3 valores
que vamos a generar. A la hora de colocarlos en los Ajustes del addon Youtube nos será de enorme ayuda, ya que son
valores con tropecientas letras y números, mayúsculas y minúsculas. Una vez colocados correctamente y que veamos que
funcionan, simplemente nos vamos a la carpeta userdata/addon_data de Kodi en nuestro Windows y nos copiamos los 3 archivos
que veremos en la carpeta del addon youtube (plugin.video.youtube) en un pen y ya podremos copiarlas en esa misma carpeta
en el resto de dispositivos que tengamos.


* Paso 1:
- Entra aquí - https://console.cloud.google.com/
- Selecciona tus preferencias de Email (Sip o Nop, lo mismo da que da lo mismo)
- Haz click en Yes (que estás de acuerdo con los Términos del Servicio (Por si estás muy aburrido y quisieras leerlos: https://console.cloud.google.com/terms)
- Haz click en Agree y Continue

Nota: Ignora cualquier oferta tipo "Sign up for a free trial..." y/o "Try Google Cloud Platform for free". No es necesario para nada, y yo de ti no lo haría, pero tú verás lo que haces, advertido quedas...

- Sigue estas opciones: Select Project > Create Project o New Project> Enter Project Name (Pon el nombre que te de la gana) > Create
- Sale una ventanita diciendo Create Project (Ahora mismo o Now). Pues pulsa ahí.






* Paso 2:

- Ahora entra aquí- https://console.developers.google.com/apis/library
- Y sigue estos pasos: YouTube APIs > Select YouTube Data API > Enable o Habilitar





* Paso 3:

- Ahora entra aquí - https://console.cloud.google.com/apis/credentials
- Y escoge: Create Credentials (Crear Credenciales) > API Key (Clave de API)
- Copiate la API Key, la necesitarás más tarde. Es privada, no se la des a nadie, ni aunque te la pida tu novi@. De hecho, a tu novi@ mucho menos.
- Y ya puedes cerrar.





* Paso 4:

- Ahora, en la misma página escoge: Create Credentials > OAuth Client ID (ID de Cliente de OAuth)
- Y ahora esto: Clickamos en Configure Consent Screen (Configurar Pantalla de Autorización)
- Nos pide que le digamos el tipo de usuario (User Type). Seleccionamos Externo (el interno además no nos deja si no somos usuarios de G Suite)
- Le damos a Crear (Create)




* Paso 5:

- Mete un nombre (el que te dé la real gana) en "Product name shown to users" o "Nombre de la Aplicación"
- Bajamos hasta abajo del todo y Click en Save (Guardar) and ESPERA que se cargue la siguiente página con el OAuth Client ID screen
- Otra vez vamos a Credenciales y escogemos Crear Credenciales y seleccionamos ID Cliente OAuth - Seleccionamos: Other (Otro) y pulsamos en Create (Crear)
- Copiáte el Client ID y el Client Secret. También los necesitarás más tarde. Y no se los des a nadie. Aunque te torturen.
- Clikea ahora en OK.





* Paso 6:

- Ahora nos vamos al addon Youtube > Ajustes > API > Habilitamos las API Keys Personales > Metemos las 3 Keys > Y OK
- Nos salimos de Kodi y entramos de nuevo. Ya debería de funcionar.



Bueno, pues si has llegado hasta aquí, ya tienes tus propias API Keys, con lo que deberían acabarse tus problemas con el addon Youtube.

Que las disfrutes !!!
